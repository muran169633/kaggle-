# NFL Project

这个目录用于存放 NFL 相关的数据分析、建模与实验文件。

## 目录说明
- `*.ipynb`：实验和分析 Notebook。
- `*.py`：训练或推理脚本。
- `model2/`、`model_st/`：模型与相关参数文件。
- `cache/`：缓存数据。
- `nfl-big-data-bowl-2026-prediction/`：比赛数据集与训练/测试输入输出文件。

## 使用建议
1. 在虚拟环境中运行 Notebook 和脚本。
2. 大体积模型文件建议单独备份。
3. 新实验建议按版本命名，便于追踪结果。
